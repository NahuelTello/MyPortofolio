Desarrollador Web | CODERHOUSE

Como desarrollador web en CoderHouse, aprendí varias habilidades y tecnologías clave que son fundamentales para crear y diseñar sitios web atractivos y funcionales. En primer lugar, aprendí HTML (Hypertext Markup Language), el lenguaje estándar utilizado para crear y estructurar el contenido de una página web. A continuación, aprendí CSS (Cascading Style Sheets), un lenguaje utilizado para dar estilo y diseño a las páginas web, incluyendo el diseño de la página, la tipografía, los colores y los efectos visuales.

Además, también aprendí SCSS (Sass), un preprocesador de CSS que te permitió escribir estilos de una manera más rápida y eficiente. También aprendí GIT, un sistema de control de versiones que te permitió colaborar con otros desarrolladores en proyectos de manera más efectiva y manejar los cambios y actualizaciones de código de manera organizada y controlada.

Finalmente, aprendí Node.js, una plataforma de desarrollo de aplicaciones que te permitió utilizar Javascript tanto en el lado del cliente como del servidor, lo que te permitió crear aplicaciones web más dinámicas e interactivas. Con estas habilidades y tecnologías, estoy preparado para comenzar a construir sitios web modernos y atractivos.
